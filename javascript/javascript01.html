01. 변수 : 데이터 저장

자바스크립트에서 변수는 값을 저장하고 참조하는데 사용합니다.
변수를 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";
    
    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
100
200
javascript

02. 변수 : 데이터 저장 + 데이터 변경

자바스크립트에서 변수에는 저장된 데이터를 변경할 수 있습니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x = 300;
    y = 400;
    z = "jquery";

    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
300
400
jquery

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

자바스크립트에서 변수는 데이터를 저장하고, 변경하고, 추가할 수 있습니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x += 300;
    y += 400;
    z += "jquery";
    console.log(x);
    console.log(y);
    console.log(z);
}
결과 확인하기
400
600
javascriptjquery

04. 상수 : 데이터 저장 + 데이터 변경(X)

자바스크립트 내에서의 상수는 데이터 저장은 가능하나, 데이터 변경은 할 수 없습니다.

05. 배열 : 데이터 저장(여러개) : 표현방법 1 : 배열 선언

배열은 여러 개의 데이터를 저장할 수 있는 데이터 구조입니다.
배열을 선언하고 데이터를 저장하는 방법은 다음과 같습니다.

▶ 대괄호([])를 사용하여 배열을 선언합니다.
▶ 각 데이터는 쉼표(,)로 구분하여 배열에 저장합니다.
▶ 배열의 각 요소는 인덱스(index)를 통해 접근할 수 있습니다. 인덱스는 0부터 시작하며, 첫 번째 요소의 인덱스는 0입니다.

{
    const arr = new Array();

    arr[0] = 100;
    arr[1] = 200;
    arr[2] = "javascript";

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}
결과 확인하기
100
200
javascript

06. 배열 : 데이터 저장(여러개) : 표현방법 2 : 배열 선언과 동시에 초기값 설정

배열을 선언하면서 동시에 초기값을 설정할 수도 있습니다.
이 방법은 배열을 선언한 후에 각 요소에 값을 할당하는 것보다 편리합니다. 

{
    const arr = new Array(100, 200, "javascript");

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}
결과 확인하기
100
200
javascript

07. 배열 : 데이터 저장(여러개) : 표현방법 3 : 리터럴 방식으로 배열 선언

배열을 선언하는 또 다른 방법은 리터럴(literal)방식을 사용하는 것입니다.
리터럴 방식은 배열을 직접 생성하는 방법입니다.
이 방법은 배열을 선언하면서 동시에 초기값을 설정하는 방법과 동일하지만, 더 간결하고 직관적입니다. 

{
    const arr = [];

    arr[0] = 100;
    arr[1] = 200;
    arr[2] = "javascript";

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}
결과 확인하기
100
200
javascript

08. 배열 : 데이터 저장(여러개) : 표현방법 4 : 리터럴 방식으로 배열 선언과 동시에 초기값 설정

배열을 리터럴 방식으로 선언하면서 동시에 초기값을 설정할 수 있습니다.
다음과 같이 배열을 리터럴 방식으로 선언하면서 각 요소의 값을 쉼표로 구분하여 초기값으로 설정할 수 있습니다.
다음의 코드에서는 arr이라는 배열을 리터럴 방식으로 선언하면서 동시에 100, 200, javascript를 초기값으로 설정하였습니다. 

{
    const arr = [100, 200, "javascript"];

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}
결과 확인하기
100
200
javascript

09. 배열 : 데이터 저장(여러개) : 표현방법 5 : push() 메서드를 이용한 배열 방법

배열에 데이터를 추가하는 방법 중 하나는 push() 메서드를 이용하는 것입니다.
push() 메서드는 배열의 끝에 요소를 추가하는 메서드입니다. 

{
    const arr = [100, 200, "javascript"];

    arr.push(400);

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
    console.log(arr[3]);
}
결과 확인하기
100
200
javascript
400

10. 배열 : 데이터 저장(여러개) : 표현방법 6 : 다차원 배열

다차원 배열은 여러 개의 배열이 중첩된 형태입니다.
다차원 배열을 이용하면 보다 복잡한 데이터를 저장하고 처리할 수 있습니다.
다차원 배열의 각 요소에 접근하려면 해당 요소의 인덱스를 사용합니다. 

{
    const arr = [100, [200, 300], ["javascript", "jquery"]];

    console.log(arr[0]);    // 100
    console.log(arr[1][0]); // 200
    console.log(arr[1][1]); // 300
    console.log(arr[2][0]); // "javascript"
    console.log(arr[2][1]); // "jquery"
}
결과 확인하기
100
200
300
javascript
jquery

11. 배열 : 데이터 저장(여러개) : 표현방법 7 : 배열구조분해할당

배열의 요소를 개별 변수에 할당할 때 배열 구조 분해 할당(Array Destructuring Assignment)을 사용할 수 있습니다.
이 방법은 배열의 각 요소를 개별 변수에 직접 할당하는 것보다 편리하고 읽기 쉽습니다. 

{
    const arr = [100, 200, "javascript"];

    const [a, b, c] = arr;

    console.log(a);     // 100
    console.log(b);     // 200
    console.log(c);     // javascript
}
{
    const arr = [100, 200, [300, 400]];
    const [a, b, [x, y]] = arr;

    console.log(a);     // 100
    console.log(b);     // 200
    console.log(x);     // 300
    console.log(y);     // 400
}
결과 확인하기
100
200
javascript

200
300

12. 배열 : 데이터 저장(여러개) : 표현방법 8 : 배열펼침연산자

배열 펼침 연산자(...)는 배열의 모든 요소를 개별적으로 나열하는 데 사용됩니다.
이는 배열의 각 요소를 개별 변수에 할당하는 배열 구조 분해 할당과 유사하지만, 변수를 명시적으로 선언하지 않습니다. 

{
    const arr1 = [100, 200, "javascript"];
    const arr2 = [300, 400, "jquery"];
    const arr3 = [...arr1, 500];

    console.log(arr1); // 100,200,javascript
    console.log(...arr1); // 100200javascript
    console.log(...arr1, ...arr2); // 100200javascript300400jquery
    console.log(...arr3)    // 100200javascript500
}
결과 확인하기
100,200,javascript
100200javascript
100200javascript300400jquery
100200javascript500

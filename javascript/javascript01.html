01. 변수 : 데이터 저장

자바스크립트에서 변수는 값을 저장하고 참조하는데 사용합니다.
변수를 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    console.log(y);
    console.log(z);
}

05. 배열 : 데이터 저장(여러개) : 표현방법 1 : 배열 선언

배열은 여러 개의 데이터를 저장할 수 있는 데이터 구조입니다.
배열을 선언하고 데이터를 저장하는 방법은 다음과 같습니다.

▶ 대괄호([])를 사용하여 배열을 선언합니다.
▶ 각 데이터는 쉼표(,)로 구분하여 배열에 저장합니다.
▶ 배열의 각 요소는 인덱스(index)를 통해 접근할 수 있습니다. 인덱스는 0부터 시작하며, 첫 번째 요소의 인덱스는 0입니다.

{
    const arr = new Array();

    arr[0] = 100;
    arr[1] = 200;
    arr[2] = "javascript";

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}
결과 확인하기
100
200
javascript

11. 배열 : 데이터 저장(여러개) : 표현방법 7 : 배열구조분해할당

배열의 요소를 개별 변수에 할당할 때 배열 구조 분해 할당(Array Destructuring Assignment)을 사용할 수 있습니다.
이 방법은 배열의 각 요소를 개별 변수에 직접 할당하는 것보다 편리하고 읽기 쉽습니다. 

{
    const arr = [100, 200, "javascript"];

    const [a, b, c] = arr;

    console.log(a);     // 100
    console.log(b);     // 200
    console.log(c);     // javascript
}
{
    const arr = [100, 200, [300, 400]];
    const [a, b, [x, y]] = arr;

    console.log(a);     // 100
    console.log(b);     // 200
    console.log(x);     // 300
    console.log(y);     // 400
}
결과 확인하기

200

12. 배열 : 데이터 저장(여러개) : 표현방법 8 : 배열펼침연산자

배열 펼침 연산자(...)는 배열의 모든 요소를 개별적으로 나열하는 데 사용됩니다.
이는 배열의 각 요소를 개별 변수에 할당하는 배열 구조 분해 할당과 유사하지만, 변수를 명시적으로 선언하지 않습니다. 

{
    const arr1 = [100, 200, "javascript"];
    const arr2 = [300, 400, "jquery"];
    const arr3 = [...arr1, 500];

    console.log(arr1); // 100,200,javascript
    console.log(...arr1); // 100200javascript
    console.log(...arr1, ...arr2); // 100200javascript300400jquery
    console.log(...arr3)    // 100200javascript500
}
결과 확인하기
100,200,javascript
100200javascript
100200javascript300400jquery
100200javascript500

20. 객체 : 데이터 저장(키와값) : 표현방법8 : 객체 + 함수가 있는 방식

객체 내에 함수를 포함하는 방식은 객체의 메소드(Method)를 정의하는 데 사용됩니다.

{
    const obj = {
            a: 100,
            b: [200, 300],
            c: "javascript",
            d: function () {
                console.log("javascript가 실행되었습니다.");
            },
            e: function () {
                console.log(obj.c + "가 실행되었습니다.");
            },
            f: function () {
                console.log(this.c + "가 실행되었습니다.");
            }
    }
        
    console.log("20. 객체 : 표현방법8");
    console.log(obj.a);     //100
    console.log(obj.b[0]);  //200
    console.log(obj.b[1]);  //300
    console.log(obj.c);     //javascript
    obj.d();     //javascript가 실행되었습니다. 함수를 실행시킬때는 ()를 사용한다.
    obj.e();    //javascript가 실행되었습니다. 같은 함수내에서 불러옴
    obj.f();    // javascript가 실행되었습니다. this와 함께 사용가능
}

21. 객체 : 데이터 저장(키와값) : 표현방법9 : 객체구조분해할당

객체 구조 분해 할당은 객체의 속성을 별도의 변수로 추출할 수 있게 하는 JavaScript의 표현식입니다.
구조 분해 할당을 사용하면 객체 내부의 키와 매칭되는 변수에 직접 값을 할당할 수 있으며, 복잡한 객체도 쉽게 다룰 수 있습니다. 

{
        const obj = {
            a: 100,
            b: 200,
            c: "javascript",
        }

        const { a, b, c } = obj;

        console.log("21. 객체 : 표현방법9");
        console.log(a);
        console.log(b);
        console.log(c);
}
결과 확인하기
100
200
javascript

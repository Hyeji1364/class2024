14. 💛filter()

const newArray = 배열.filter(function(element, index, array) {

return 조건; // 조건이 true일 때 요소를 새 배열에 포함
});

element: 배열에서 현재 처리 중인 요소입니다.

index: 배열에서 현재 처리 중인 요소의 인덱스입니다.

array: filter()가 호출된 배열입니다.

  • 이 메서드는 주어진 함수 조건에 맞는 모든 요소를 새로운 배열로 반환합니다.
  • 콜백 함수를 인자로 받으며, 이 콜백 함수는 각 요소를 평가하여 'true' 또는 'false'를 반환합니다.
  • true를 반환하는 요소만이 새 배열에 포함됩니다.
  • filter()는 원본 배열을 변경하지 않으며, 조건에 맞는 요소들로 구성된 새로운 배열을 반환합니다.
{

}

위의 예시에서는 숫자 배열에서 짝수만 필터링하여 새로운 배열을 생성하는 방법을 보여줍니다.
이 코드는 배열 numbers의 각 요소를 콜백 함수에 전달하고, 짝수인지 확인합니다.
짝수일 경우 true를 반환하며, 이 때문에 해당 요소는 evenNumbers 배열에 포함됩니다.

16. 💛💛💛map()

const newArray = 배열.map(function(element, index, array){

return 새로운 값;
});

  • 이 메서드는 주어진 함수를 호출하여 각 요소에 대한 결과를 모아 새로운 배열을 만듭니다.
  • 이 메서드는 원본 배열의 모든 요소를 하나씩 처리하여 그 결과를 새 배열의 해당 위치에 저장합니다.
  • map()은 배열의 길이를 변경하지 않고, 원본 배열은 수정하지 않으면서 각 요소에 대해 주어진 함수를 실행한 결과로 구성된 새 배열을 반환합니다.
  • map() 메서드는 데이터를 변환하거나 수정하여 새로운 형태의 데이터를 생성할 때 매우 유용합니다.
  • 예를 들어, 객체 배열에서 특정 키의 값을 수정하거나 새로운 키를 추가하는 등의 작업에 자주 사용됩니다.
{
const numbers = [1, 2, 3, 4, 5];
const squaredNumbers = numbers.map(function(number) {
    return number * number; // 각 숫자를 제곱
});
console.log(squaredNumbers); // 출력: [1, 4, 9, 16, 25]
}

위의 예시에서는 숫자 배열의 각 요소를 제곱하여 새로운 배열을 생성하는 방법을 보여줍니다.
이 코드는 numbers 배열의 각 요소에 대해 제곱 연산을 수행하고, 그 결과를 squaredNumbers라는 새 배열에 저장합니다. 결과적으로, 각 숫자의 제곱값을 요소로 하는 새 배열이 출력됩니다.

21. 💛💛join()

const newString = 배열.join(구분자);

💡 구분자 : 배열의 각 요소를 연결할 때 사이에 끼워넣을 문자열입니다. 이 인자는 선택적이며, 생략하면 요소들 사이에 쉼표가 들어갑니다.

  • 메서드는 배열의 요소를 연결하여 하나의 문자열로 반환합니다.
  • 이 메서드는 배열 요소 사이에 삽입할 문자열을 인자로 받으며, 주어지지 않은 경우 기본적으로 쉼표(,)를 사용합니다.
  • join()은 원본 배열을 변경하지 않고, 요소들을 연결한 새로운 문자열을 반환합니다.
  • 'join()' 메서드는 배열의 요소들을 하나의 문자열로 결합할 때 매우 유용합니다.
  • 이를 통해 간단하게 배열 데이터를 텍스트 형태로 출력하거나 파일로 저장할 데이터를 준비할 수 있습니다.
  • 사용자 정의 구분자를 설정함으로써 다양한 형식의 데이터를 생성할 수 있으며, 이는 로그 파일 생성이나 데이터 전송 등에서 특히 유용하게 사용될 수 있습니다.
{
// 01. 간단한 배열을 문자열로 변환:
const fruits = ['apple', 'banana', 'mango'];
const resultString = fruits.join(); // 구분자를 지정하지 않았기 때문에 쉼표가 사용됨
console.log(resultString); // 출력: "apple,banana,mango"

// 02. 쉼표 대신 다른 구분자 사용:
const fruits = ['apple', 'banana', 'mango'];
const resultString = fruits.join(' - '); // 구분자로 하이픈과 공백 사용
console.log(resultString); // 출력: "apple - banana - mango"

// 03. 빈 문자열을 구분자로 사용:
const letters = ['H', 'e', 'l', 'l', 'o'];
const word = letters.join(''); // 구분자로 빈 문자열 사용
console.log(word); // 출력: "Hello"

}

07. 💛reduce()

const result = 배열.reduce(function(accumulator, currentValue, currentIndex, array) {
// 처리 로직
return 새로운 누적 값;
}, 초기값);

accumulator: 이전 콜백 호출의 반환 값이며, 첫 번째 호출에서는 초기값입니다.
currentValue: 배열에서 현재 처리 중인 요소입니다.
currentIndex (선택적): 배열에서 현재 처리 중인 요소의 인덱스입니다.
array (선택적): reduce() 함수가 호출된 배열입니다.
초기값 (선택적): accumulator의 첫 번째 값으로 사용됩니다. 이 값이 제공되지 않으면 배열의 첫 번째 요소가 초기값으로 사용됩니다.

  • 메서드는 배열의 각 요소에 주어진 리듀서 함수를 실행하고, 하나의 결과값을 반환합니다.
  • JavaScript에서 배열의 각 요소를 하나씩 처리하여 단일 결과 값(누적 결과)을 생성하는 데 사용됩니다.
  • 이 함수는 배열을 "축소"하여 하나의 값으로 합치는 작업에 적합합니다.
  • 배열의 요소를 합산, 평균 계산, 객체 생성 등 다양한 방식으로 데이터를 집계할 때 유용합니다.
{
// 01. 숫자 배열의 총합 계산하기:
const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce(function(accumulator, current) {
    return accumulator + current;
}, 0);  // 초기값으로 0을 설정
console.log(sum); // 출력: 15

// 02. 객체 배열에서 특정 속성의 값 합산하기:
const items = [
    { name: "apple", quantity: 4 },
    { name: "banana", quantity: 2 },
    { name: "orange", quantity: 5 }
];

const totalItems = items.reduce(function(accumulator, item) {
    return accumulator + item.quantity;
}, 0);  // 초기값으로 0을 설정

console.log(totalItems); // 출력: 11
}

📍 01. 숫자 배열의 총합 계산하기
이 코드는 numbers 배열의 모든 숫자를 합하는 작업을 수행합니다.
accumulator는 누적 합계를 저장하며, current는 현재 숫자를 나타냅니다.
초기값으로 0이 설정되어 있으므로 계산은 0에서 시작합니다.

📍 02. 시작 인덱스와 끝 인덱스 사용하기
이 예시는 각 항목의 quantity 값을 합산하여 총 합을 계산합니다.
reduce() 함수는 각 요소의 quantity를 누적 합계에 더해 나갑니다.

결과 확인하기

07. 💛reverse()

const reversedArray = 배열.reverse();

  • 메서드는 배열 요소의 순서를 반대로 하고, 그 결과로 변경된 배열을 반환합니다.
  • 이 메서드는 배열의 첫 번째 요소와 마지막 요소를 서로 바꾸고, 두 번째 요소와 끝에서 두 번째 요소를 서로 바꾸는 식으로 계속해서 배열의 순서를 뒤집습니다.
  • 중요한 점은 reverse()가 원본 배열을 직접 수정한다는 것입니다.
  • 따라서 이 메서드를 호출하면 원래의 배열이 변경되며, 변경된 배열 자체가 반환됩니다.
  • reverse() 메서드는 배열의 순서를 빠르게 뒤집을 수 있어서 데이터의 순서를 변경해야 할 때 유용하게 사용됩니다.
{
// 01. 문자열 배열 뒤집기:
const letters = ['a', 'b', 'c', 'd'];
const reversedLetters = letters.reverse();

console.log(reversedLetters); // 출력: ['d', 'c', 'b', 'a']
console.log(letters); // 출력: ['d', 'c', 'b', 'a'] (원본 배열도 변경됨)

// 02. 숫자 배열 뒤집기:
const numbers = [1, 2, 3, 4, 5];
numbers.reverse();

console.log(numbers); // 출력: [5, 4, 3, 2, 1]
}

📍01. 문자열 배열 뒤집기
이 예시에서는 letters 배열의 요소 순서를 뒤집어 reversedLetters에 저장하고 있습니다.
reverse()는 원본 배열 letters도 변경하기 때문에, letters 배열도 뒤집힌 상태로 출력됩니다.

📍02. 시작 인덱스와 끝 인덱스 사용하기
이 코드는 numbers 배열의 숫자 순서를 뒤집습니다.
배열을 뒤집은 후에는 원본 배열 numbers가 업데이트되어 뒤집힌 상태로 출력됩니다.

결과 확인하기
['d', 'c', 'b', 'a'] ['d', 'c', 'b', 'a'] (원본 배열도 변경됨) [5, 4, 3, 2, 1]

07. 💛💛find()

"문자열".find(찾을_문자열, 시작_인덱스, 끝_인덱스)

  • 메서드는 주어진 판별 함수를 만족하는 첫 번째 요소의 값을 반환합니다.
  • find()메서드는 문자열에서 특정 부분 문자열을 찾고 그 위치를 알려주는 python의 문자열 메서드입니다.
  • 이 메서드를 사용하면 부분 문자열이 시작되는 첫 번째 위치의 인덱스를 알 수 있습니다.
  • 찾고자 하는 부분 문자열이 없다면 '-1'을 반환합니다.
  • 이 메서드는 대소문자를 구분하므로, 정확한 대소문자를 사용하여 검색어를 입력해야 합니다.
  • 문자열에서 처음 발견되는 부분 문자열의 인덱스만을 반환합니다.
{
// 01. 기본 예시
s = "Hello, world!"
print(s.find("world"))  # 출력: 7
print(s.find("Python")) # 출력: -1  

// 02. 시작 인덱스와 끝 인덱스 사용하기
s = "Look over there, over here!"
print(s.find("over", 5))       # 출력: 17
print(s.find("over", 9, 13))   # 출력: -1
}

📍01. 기본예시
여기서 'world'는 문자열 's'의 7번 인덱스에서 시작하기 때문에 '7'을 반환합니다.
'Python'은 문자열 's'에 존재하지 않기 때문에 '-1'이 반환됩니다.

📍02. 시작 인덱스와 끝 인덱스 사용하기
두 번째 호출은 인덱스 5부터 13까지의 범위에서 'over'문자열이 포함되어 있지 않으므로 '-1'이 출력됩니다.

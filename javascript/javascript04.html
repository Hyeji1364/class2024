01. if문

if문은 프로그래밍에서 조건에 따라 다른 코드 블록을 실행하도록 하는 기본적인 조건문입니다.
조건식이 참(true)일 때만 특정 코드 블록을 실행하게 됩니다. 

{
    // true, "문자열" , 숫자 1, 2, 문자열 "1" , "2" ,배열 : [], 객체 : {}
    // false, ""(빈문자열), 0, null, undefined

    if (0) {
        console.log("실행되었습니다.(true)");
    } else {
        console.log("실행되었습니다.(false)");
    }
}

02. 다중 if문

다중 if문은 여러 조건을 순차적으로 검사하여 각 조건에 따라 다른 작업을 수행하도록 하는 구조입니다.
이는 복잡한 결정 구조를 구현할 때 매우 유용합니다.
여러 조건에 따라 조건문을 설정합니다. 조건에 계속 설정할 수 있으며, 마지막에는 조건식을 쓰지 않는 else를 사용할 수 있습니다. 

{
        const num = 100;

        if (num == 90) {
            console.log("실행되었습니다.(90)");
        } else if (num == 100) {
            console.log("실행되었습니다.(100)");
        } else if (num == 110) {
            console.log("실행되었습니다.(110)");
        } else if (num == 120) {
            console.log("실행되었습니다.(120)");
        } else if (num == 130) {
            console.log("실행되었습니다.(130)");
        } else {
            console.log("실행되었습니다.");
        }
}
결과 확인하기
실행되었습니다.(100)

03. 중첩 if문

중첩 if문은 조건문 안에 또 다른 조건문이 있는 경우를 말합니다. 이를 통해 더 세분화된 조건을 설정할 수 있습니다. 

{
        const num = 100;

        if (num == 100) {
            console.log("실행되었습니다.1");
            if (num == 100) {
                console.log("실행되었습니다.2");
                if (num == 100) {
                    console.log("실행되었습니다.3")
                }
            }
        } else {
            console.log("실행되었습니다.4")
        }
}
결과 확인하기
실행되었습니다.1
실행되었습니다.2
실행되었습니다.3

04. if문 생략 & 삼항 연산자

Javascript에서는 조건이 참일때만 실행되어야 하는 단일 명령어의 경우, if문을 생략하고 조건과 명령어를 한 줄에 작성할 수 있습니다.
삼항연산자는 조건에 따라 두 가지의 경우 중 하나를 선택해야 할 때 사용합니다.
" 조건 ? 참일 때의 값 : 거짓일 때의 값 "의 형태로 사용됩니다. 

{
        const num = 100;        // 조건

        if (num == 100) {
            console.log("true");
        }

        // 생략

        if (num == 100) console.log("true");

        if (num == 100) {
            console.log("true");
        } else {
            console.log("false");
        }

        // 삼항 연산자(조건식 연산자)
        num == 100 ? console.log("true") : console.log("false");
}

05. switch문

Javascript에서 switch문은 특정 변수의 값에 따라 다양한 경우(case)를 실행할 수 있는 제어문입니다.
여러 if문을 사용하는 것보다 가독성이 좋고 관리하기 쉬운 경우가 많습니다.

{
        const num = 100;

        switch (num) {
            case 90:
                console.log(first)("실행되었습니다.(90)");
                break;
            case 95:
                console.log("실행되었습니다.(95)");
            case 100:
                console.log("실행되었습니다.(100)");
                break;
            case 105:
                console.log("실행되었습니다.(105)");
                break;
            default:
                console.log("실행되었습니다.");
        }
}

06. while문

JavaScript에서 while문은 조건이 참(true)인 동안 코드 블록을 반복해서 실행하는 루프 제어문입니다.
조건이 거짓(false)이 되면 반복이 멈춥니다.
아래의 예시에서는 num변수가 10보다 작거나 같은 동안, 콘솔에 num의 값을 출력하고 count를 1씩 

{
        let num = 1;                // 초깃값;

        while (num <= 10) {         // 조건식;
            console.log(num);       // 실행문
            num++                   // 증감값;
        }
}

07. do while문

do while문은 JavaScript에서 실행하는 반복문 중 하나입니다.
이 반복문은 조건을 검사하기 전에 코드 블록을 최소 한 번 실행하고, 그 후 조건이 참인 동안 코드 블록을 반복 실행합니다.
do while문은 조건이 거짓이 되기 전까지 코드 블록을 반복해서 실행합니다.
다른 반복문과 달리, do while문은 조건을 검사하기 전에 코드 블록을 최소 한 번 실행합니다.
이는 조건에 관계없이 코드 블록이 최소 한 번은 실행되어야 할 때 유용합니다. 

{
        let num = 1;   // 초깃값;

        do {
            console.log(num);       // 실행문
            num++;      // 증감값;
        } while (num <= 10);      // 조건식
}

08. for문

for문은 반복 횟수가 명확할 때 주로 사용됩니다. for문은 초기화, 조건 검사, 그리고 반복 실행될 때마다 수행될 업데이트를 한 줄에 포함하여 반복 작업을 간결하게 표현할 수 있습니다.
배열이나 객체와 같은 데이터 구조를 순회할 때, 또는 특정 조건을 만족할 때까지 반복 작업을 수행할 때 매우 유용합니다.
코드를 간결하게 유지하면서 반복 작업을 효율적으로 처리할 수 있습니다. 

{
    for (let i = 1; i <= 10; i++) {     // 초깃값; 조건식; 증감값;
        console.log(i);
    }
}

09. 중첩 for문

중첩 for문은 하나의 for문 안에 다른 for문이 포함되어 있는 구조를 말합니다.
이러한 구조는 다차원 배열을 다루거나, 복잡한 데이터 구조에 대한 연산을 수행할 때 유용하게 사용됩니다.
특히, 각각의 내부 루프가 외부 루프의 각 반복에 대해 실행될 때, 중첩 for문은 강력한 반복 처리 능력을 발휘합니다.
외부 루프가 한 번 반복할 때마다 내부 루프는 자신의 반복을 처음부터 끝까지 실행합니다.
내부 루프가 모든 반복을 마친 후에야 외부 루프가 다음 반복으로 넘어갈 수 있습니다. 

{
        for (let i = 1; i <= 10; i++) {
            for (let j = 1; j <= 10; j++) {
                console.log(i, j);
            }
        }
}

10. break문

프로그래밍에서 루프(loop)나 스위치(switch)문의 실행을 즉시 중단시키는 역할을 합니다.
주로 반복문 내에서 특정 조건을 만족할 때 더 이상의 반복을 중지하고 반복문을 벗어나고자 할 때 사용됩니다.

{
        for (let i = 1; i <= 10; i++) {
            if (i == 5) {
                break;
            }
            console.log(i)
        }
}

11. continue문

continue문은 for, while, do while 반복 문 내에서 사용될 때, continue문을 만나면 현재 반복의 나머지 부분을 건너뛰고 반복문의 다음 반복으로 진행합니다.
또한 조건에 따른 실행을 건너뛰기하는 역할도 수행합니다. 특정 조건을 만족하는 경우에만 일부 코드를 건너뛰고 싶을 때 유용하게 사용됩니다.
아래의 코드는 JavaScript로 작성된 반복문의 예시입니다.
코드를 실행하면 1부터 10까지의 숫자를 출력하는데, continue문이 사용되어 숫자 5일때의 출력을 건너뛰고 출력됩니다. 

{
        for (let i = 1; i <= 10; i++) {
            if (i == 5) {
                continue;
            }
            console.log(i)
        }
}

06.배열 : 데이터 불러오기: forEach()문

배열의 forEach문은 배열의 각 요소에 대해 함수를 호출하는 메서드입니다.

    arr.forEach(function (element, index) {
        console.log(index);
    });
    //03
    arr.forEach(function (element, index, array) {
        console.log(array);
    });
}

07.배열 : 데이터 불러오기: for of

배열의 for of문은 배열의 각 요소를 순회하면서 각 요소에 접근할 수 있는 방법을 제공하는 구문입니다.
for of문은 ES6에서 새롭게 추가된 구문으로, 기존의 for문보다 더 간결하고 직관적으로 배열의 요소를 순회할 수 있습니다.

10. 배열: 데이터 불러오기: filter() // 함수 안의 함수

{
    const num = [1, 2, 3, 4, 5, 6, 7, 8, 9];

    console.log("10++++++");
    num.filter(function (element) {
        console.log(element);
    });                                 // 

    num.filter(function (element, index) {
        console.log(element);
    });

    num.filter(function (element, index, array) {
        console.log(element);
    });

    const arr = num.filter(function (element) {
        return element > 5;
    });

    const arr1 = num.filter((element) => {
        return element > 5;
    });

    const arr2 = num.filter(element => {
        return element > 5;
    });

    const arr3 = num.filter(e => e > 5);    //return 생략

    console.log(arr)
}

14. 객체: 데이터 불러오기: Object.keys()

17. 객체: 데이터 불러오기: Object.assign()

Object.assign()은 자바스크립트에서 객체의 속성을 복사하거나 병합할 때 사용하는 함수입니다.
원본 객체의 속성을 대상 객체에 복사하거나, 여러 개의 객체를 병합하여 새로운 객체를 생성할 수 있습니다. 아래의 함수에서는 obj1과 obj2의 속성을 병합하여 obj3를 생성한 결과입니다.
obj3에는 obj1과 obj2의 모든 속성이 포함되어 있습니다. 

{
    {
        const obj1 = {
            a: 100,
            b: 200,
        }

        const obj2 = {
            c: 300,
            d: 400,
        }
        const obj3 = Object.assign(obj1, obj2);

        console.log(obj3);
    }
}
결과 확인하기
{ a: 100, b: 200, c: 300, d: 400 }

18. 객체: 데이터 불러오기: hasOwnProperty()

hasOwnProperty() 함수는 객체가 특정 속성을 소유하고 있는지 확인하는 데 사용되는 함수입니다.
아래의 예시에서 obj 객체가 "a", "b", "c" 속성을 소유하고 있으므로 각각 true를 반환합니다.
"d" 속성은 obj 객체에 존재하지 않으므로 false를 반환합니다.
in 연산자는 객체가 특정 속성을 소유하고 있는지 확인하는 데 사용됩니다.
아래의 코드에서는 "a", "b", "c" 속성이 obj 객체에 존재하므로 각각 true를 반환합니다.
"d" 속성은 obj 객체에 존재하지 않으므로 false를 반환합니다. 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }

    console.log(obj.hasOwnProperty("a"));   // true
    console.log(obj.hasOwnProperty("b"));   // true
    console.log(obj.hasOwnProperty("c"));   // true
    console.log(obj.hasOwnProperty("d"));   // false

    // in 연산자
    console.log("a" in obj);         // true
    console.log("b" in obj);         // true
    console.log("c" in obj);         // true
    console.log("d" in obj);         // false
}

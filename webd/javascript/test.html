01. 1부터 100까지 출력하기(for문, while, do while)

for문과 while문, do while문을 사용하여 1부터 100까지 실행하는 예시입니다. 

{
    for (let i = 0; i <= 100; i++) {
        console.log(i);
    }

    // 1,2,3,4 ... 100
}
{
    let num = 0;        // 초깃값 설정
    while (num <= 100) {
        console.log(num);
        num++;
    }
}
{
    let num = 0;
    do {
        console.log(num);
        num++;
    } while (num <= 100);
}
결과 확인하기
1~100
1~100
1~100

02. 0부터 100까지 짝수만 출력하기(for문, if문)

02-1. 아래의 코드는 for문을 사용하여 i를 0부터 100까지 1씩 증가시키면서 반복합니다.
i를 2로 나눈 나머지가 0인 경우, i가 짝수인 경우에만 console.log(i)를 실행하여 i를 출력합니다.
02-2. 짝수의 경우에는 i를 2로 나눈 나머지가 1인 경우, 즉 i가 홀수인 경우에 console.log(i)를 실행하여 i를 출력합니다. 

{
    // 02-1.
        for (let i = 0; i <= 100; i++) {
        if (i % 2 == 0)
            console.log(i)
    }
    //02-2. 0부터 100까지 홀수만 출력하기(for문, if문)

    for (let i = 0; i <= 100; i++) {
        if (i % 2 == 1)
            console.log(i)
    }
}
결과 확인하기

03. 1부터 100까지 짝수는 파란색, 홀수는 빨간색 출력하기

for문을 사용하여 1부터 100까지 숫자를 반복하여 출력하면서, i를 2로 나눈 나머지가 0일때, 즉 짝수일때는 span tag안에 색을 지정하여 파란색으로 나오게하고, 그렇지 않으면 (홀수일 경우) 빨간색으로 표시하는 예시입니다.
document.write()함수를 통해 HTML문서에 직접 쓰기를 수행합니다 

{
    for (let i = 1; i <= 100; i++) {
        if (i % 2 == 0) {
            document.write("" + i + "");
        } else {
            document.write("" + i + "");
        }
    }
}

04. 중첩 for문을 이용해서 구구단 출력하기

외부 for문: i 변수를 2부터 시작하여 9까지 1씩 증가시키면서 반복합니다.
이 반복문은 구구단의 '단'을 나타냅니다.
내부 for문 : j 변수를 1부터 시작하여 9까지 1씩 증가시키면서 반복합니다.
이 반복문은 각 단의 곱셈을 1부터 9까지 출력합니다.
console.log(): 현재 단(i)과 곱하는 수(j), 그 결과값( i * j )을 문자열로 결합하여 콘솔에 출력합니다. 

{
    for (let i = 2; i <= 9; i++) {
        for (let j = 1; j <= 9; j++) {
            console.log(i + " * " + j + " = " + i * j)
        }
    }
}
결과 확인하기
2 * 1 = 2
2 * 2 = 4
2 * 3 = 6
.... 9 * 9 = 81

05. 중첩 for문을 이용해서 8단 출력하기

외부 for문: i 변수가 8에서 시작하여 8까지(즉, 한 번만) 반복합니다. 이 반복문은 구구단의 '단'을 설정합니다.
내부 for문: j 변수를 1부터 시작하여 9까지 1씩 증가시키면서 반복합니다. 이 반복문은 8단의 각 곱셈을 수행합니다.
console.log(): 현재 단(i)과 곱하는 수(j), 그리고 그 결과값(i * j)을 문자열로 결합하여 콘솔에 출력합니다. 

{
    for (let i = 8; i <= 8; i++) {
        for (let j = 1; j <= 9; j++) {
            console.log(i + " * " + j + " = " + i * j)
        }
    }
}
결과 확인하기
1~100
1~100
1~100
